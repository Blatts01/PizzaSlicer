
Pizza2 - v3 yolov4tiny-320
==============================

This dataset was exported via roboflow.ai on October 23, 2020 at 12:33 PM GMT

It includes 833 images.
Pizza-Pepperoni are annotated in COCO format.

The following pre-processing was applied to each image:
* Resize to 320x320 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* Random exposure adjustment of between -12 and +12 percent
